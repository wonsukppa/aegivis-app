# [기술초안] 적응형 배수 프로파일링 및 이상 유입 감지 알고리즘

**작성**: 기술연구소 박지훈 소장 (AI Persona)
**배경**: 지하차도마다 다른 배수 능력과 지형적 특성을 AI가 스스로 학습하여 맞춤형 자율 방재를 구현함.

---

## 1. 알고리즘 개념: "Evacuation Curve Learning"
모든 지하차도는 고유의 배수 곡선(Drainage Curve)을 가집니다. 본 알고리즘은 평상시 강우 데이터를 축적하여 해당 구간의 정상적인 배수 속도를 프로파일링하고, 이를 실시간 데이터와 대조합니다.

## 2. 핵심 수식 모델 (Conceptual)

### **A. 배수 효율 계수 (Drainage Efficiency Factor, η)**
$$\eta = \frac{\Delta SumpLevel + PumpCapacity}{LocalRainfall \times CatchmentArea}$$
*   **η**: 해당 지하차도의 실시간 배수 효율.
*   값이 1보다 낮아질 경우, 배수구 막힘이나 펌프 고장 등 '이상 상황'으로 즉시 인지.

### **B. 침수 가속도 지수 (Flood Acceleration Index, α)**
$$\alpha = \frac{d^2(SurfaceLevel)}{dt^2}$$
*   수위의 단순 상승이 아닌, **상승의 가속도**를 측정합니다. 
*   가속도가 양수(+)로 급증할 경우, 기상 데이터와 결합하여 "N분 후 차량 통행 불가능" 지점을 정밀 예측합니다.

## 3. 적응형 학습 프로세스 (Self-Learning Steps)

1.  **초기 설정 (Cold Start)**: 설계 도면상의 배수 용량을 기준으로 기본 임계치 설정.
2.  **데이터 적재 (Data Ingestion)**: 기상청 API 및 현장 센서를 통해 강우량 대비 집수정 수위 변화를 지속적으로 태깅.
3.  **프로파일링 (Profiling)**: 강수 강도별 정상 배수 속도를 '디지털 배수 프로필'로 저장.
4.  **이상 감지 (Anomaly Detection)**: 실시간 데이터가 저장된 프로필에서 15% 이상 벗어날 경우(예: 배수구 낙엽 막힘), 수위가 낮더라도 **'예비 경보'** 발령.

## 4. 기대 효과 (Patent Value)
*   **현장 맞춤형 방재**: 표준화된 수치가 아닌, "우리 동네 지하차도"에 딱 맞는 초정밀 경보 가능.
*   **사전 장애 인지**: 물이 차기 전, 배수 시스템의 기능 저하를 먼저 알아내어 현장 점검 유도.
