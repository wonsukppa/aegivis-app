# 이지비스 3.0 (Aegivis) - 모바일 앱 PWA 프로토타입

지하차도 침수 예측 및 AI 통합 관제를 위한 필드 요원용 고성능 모바일 PWA입니다.

## 주요 기능
- **실시간 대시보드**: 서울시 25개 자치구의 지하차도 수위 및 위험 수준 실시간 모니터링.
- **AI 예측 엔진**: 다각도 센서 데이터를 기반으로 침수 전조를 사전에 감지하고 최적의 출동 타이밍 제안.
- **현장 인증 및 보고**: QR 코드를 통한 도착 인증 및 단계별 증거 사진 수집/최종 보고 자동화.
- **프리미엄 UI/UX**: 글래스모피즘 기반의 세련된 디자인과 부드러운 페이지 전환 애니메이션.

## 기술 스택
- **Core**: HTML5, Vanilla JavaScript
- **Styling**: Tailwind CSS, FontAwesome 6
- **Architecture**: Single Page Application (SPA) structure with PWA support

## 설치 및 실행 방법
1. 본 리포지토리를 클론합니다.
2. `index.html`을 브라우저에서 실행합니다.
3. 모바일 브라우저의 '홈 화면에 추가' 기능을 통해 PWA로 설치하여 사용할 수 있습니다.
